# <p align="center">Minutely</p>

Minutely is a set of Open Source projects which empower users to use and deploy
video conferencing platforms with state-of-the-art video quality and features.

<hr />

<p align="center">
<img src="images/minutely-logo.png" width="900" />
</p>

<hr />

Amongst others here are the main features Minutely offers:

* Support for all current browsers
* Mobile applications
* Web and native SDKs for integration
* HD audio and video
* Content sharing
* Raise hand and reactions
* Chat with private conversations
* Polls
* Virtual backgrounds

And many more!

## Using Minutely

Using Minutely is straightforward, as it's browser based.
